{Habilidades}

HTML

Como muitos iniciantes no mundo de Desenvolvimento Web, o meu primeiro contato foi com a linguagem de marcação HTML. Uma linguagem de marcação a qual me chamou muita atenção para trilhar esse caminho que a Web me proporciona. Sou capaz de criar páginas acessíveis e com muitos detalhes.

CSS

Usar o Css para criar páginas muito atraentes parece ser algo muito complexo no início, porém, utilizando o Css, posso fazer páginas responsivas e com muitos efeitos/animações bem atraentes para uma melhor experiência do usuário.

JAVASCRIPT

Para criar um site dinâmico passando toda uma lógica por trás, o Javascript tem sido o carro chefe. Sou capaz de implementar lógicas a nível iniciante, utilizando todo o básico para intereções com o usuário e lhe proporcionando uma melhor experiência rica em interações.

BOOTSTRAP

Pensando em agilidade, o Bootstrap pode ser um grande aliado quando queremos soluções mais rápidas. Gosto muito de utiliza-lo em meus projetos para aumentar a produtividade em meus códigos.

JQUERY

Uma biblioteca com muitas opções de acesso pelo DOM (Document Object Model), possuo conhecimento para utilizar a biblioteca Jquery alguns de meus projetos para melhorar a usabilidade e interatividade da página.

PHP

Tendo em mente o backend de uma aplicação, nada melhor do que utilizar uma poderosa linguagem do lado do servidor, com o PHP consigo desenvolver soluções a nível iniciante. Por ser uma linguagem que venho estudando a pouco tempo, tenho me dedicado a fazer projetos em cursos para poder implementar em meus próprios projetos pessoais.

GIT

Com certeza a utilização do Git é muito importante quando se trata de várias ideias e projetos, possuo conhecimento a respeito de versionamento de código para trabalhar em equipe

SQL

Uma base de dados é algo crucial quando se trata de informações extremamente valiosas para uma melhor usabilidade do projeto. Sou capaz de criar tabelas e fazer consultas a nível iniciante, claro, sempre estudando e procurando melhorar a eficiência e a segurança dos bancos de dados em meus projetos.
